💡 Pro Tips

  • Local Server: Use a local web server for best results (VS Code Live Server, Python http.server, etc.)
  • Browser DevTools: Press F12 to open DevTools and check the Console for any errors
  • Cache: Use Ctrl+F5 (or Cmd+Shift+R on Mac) for hard refresh if changes don't appear
  • Mobile: Test responsive design by resizing browser window

⚠️ Troubleshooting

Components not loading?
  • Make sure you're using a local web server (file:// protocol may block fetch requests)
  • Check browser console for CORS errors
  • Verify file paths are correct
Sidebar not highlighting correctly?
  • Wait a moment for components to load
  • Check that JavaScript is enabled
  • Look for errors in browser console
Navigation not working?
  • Ensure all files are in correct folders
  • Check that href values match actual filenames
  • Verify no typos in file names
